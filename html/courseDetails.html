N5Course Overview

📅 Course Overview

  • Session Duration: 3 hours per session

  • Total Weekly Sessions: 3 sessions per week

  • Weekly Course Hours: 9 hours (3 sessions x 3 hours each)

  • Total Course Hours: 160 hours

  • Total Weeks: 18 weeks

  • Course Duration: 4 months 5 days

📚 Learning Focus and Tools

  • Kanji Focus: Basic kanji (approximately 110 characters)

  • Learning Tools:

    • RolePlay: Practice your language learning milestones through interactive role-playing scenarios.

    • Multimedia: Visual information (videos, interactive slides) is used to support teachings.

    • Audio Lessons: Audio materials for improving listening skills.

    • Apps: Enjoyable language practice makes learning easier.

📝 Assessment Structure

  • Class Tests: 25 (short, regular assessments)

  • Main Tests: 5 (comprehensive evaluations)

  • Class Reviews: 2 (progress reviews, feedback, and clarification)

  • Mock (MOC) Tests: 3 (practice tests simulating the final exam)

📘 Book Used

  • Book Title: Minna no Nihongo 1

🎓 Certification Details

  • Certification Level: Basic Japaneses (N5)

  • Course Title: Basic Japaneses (N5) (Based on Minna no Nihongo 1)

  • Certification Objective:

    • Basic Grammar: Understanding fundamental sentence structures.

    • Vocabulary: Building a core set of everyday words and phrases.

    • Kanji: Recognition and understanding of approximately 110 basic characters.

    • Listening: Build up listening skills and basic conversation skills.

    • Speaking: Build up basic conversation skills.

Upon successful completion of the course, participants will be awarded a Beginner (N5) certification in Japanese.
